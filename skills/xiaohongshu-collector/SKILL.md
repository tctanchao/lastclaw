# 小红书爆款采集 📕

> 采集小红书笔记数据，分析爆款内容，助力内容创作

## 功能特性

### ✅ 核心功能

- **关键词搜索**: 搜索指定关键词的笔记
- **数据采集**: 提取标题、内容、点赞数、收藏数、评论数
- **爆款识别**: 自动识别高互动笔记（点赞>1000）
- **数据导出**: 导出为 Excel/CSV 格式
- **博主监控**: 监控指定博主的最新笔记

### 🎯 使用场景

| 场景 | 说明 |
|------|------|
| 内容创作 | 找爆款选题，分析热门内容 |
| 竞品分析 | 监控竞品账号，学习爆款方法 |
| 市场调研 | 了解用户需求，发现热门话题 |
| 账号运营 | 优化内容策略，提升互动率 |

## 安装

```bash
openclaw skill install xiaohongshu-collector
```

## 配置

在 `~/.openclaw/config/skills/xiaohongshu-collector.json` 中配置：

```json
{
  "apiKey": "your-api-key",
  "searchLimit": 50,
  "exportFormat": "excel",
  "minLikes": 1000
}
```

## 使用示例

### 搜索笔记

```
/小红书采集 搜索 关键词：护肤心得
```

### 分析爆款

```
/小红书采集 分析 关键词：减肥餐
```

### 导出数据

```
/小红书采集 导出 文件名：爆款笔记.xlsx
```

### 监控博主

/小红书采集 监控 博主：李佳琦
