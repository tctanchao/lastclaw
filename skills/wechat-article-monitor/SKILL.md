# 公众号文章监控 📰

> 监控微信公众号最新文章，实时推送更新提醒

## 功能特性

### ✅ 核心功能

- **添加公众号**: 支持通过公众号名称或 ID 添加
- **文章采集**: 获取最新 N 篇文章
- **实时监控**: 检测新文章发布并推送通知
- **数据存储**: 本地保存历史记录，避免重复提醒
- **文章导出**: 导出为 Excel/Markdown 格式
- **定时检查**: 可配置检查间隔（默认 1 小时）

### 🎯 使用场景

| 场景 | 说明 |
|------|------|
| 内容学习 | 监控行业大号，第一时间学习新内容 |
| 竞品分析 | 跟踪竞品公众号，分析发布策略 |
| 企业监控 | 监控公司公众号，及时获取行业动态 |
| 账号运营 | 监控自己的公众号，确认文章发布成功 |

## 安装

```bash
openclaw skill install wechat-article-monitor
```

## 配置

在 `~/.openclaw/config/skills/wechat-article-monitor.json` 中配置：

```json
{
  "checkInterval": 3600,
  "notifyEnabled": true,
  "maxArticles": 10,
  "apiKey": "your-wechat-api-key"
}
```

### 配置说明

| 参数 | 默认值 | 说明 |
|------|----------|------|
| checkInterval | 3600 | 检查间隔（秒），默认 1 小时 |
| notifyEnabled | true | 是否启用通知推送 |
| maxArticles | 10 | 每次获取的最大文章数 |
| apiKey | - | 微信 API Key（可选） |

⚠️ **使用限制**
- 免费版：最多监控 3 个公众号
- 付费版：无限监控，高级分析

⚠️ **合规提示**
- 请遵守微信平台规则
- 仅用于个人学习和研究
- 不要用于商业爬虫
