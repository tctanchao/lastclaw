# 公众号文章监控使用指南

## 快速开始

```bash
# 安装技能
openclaw skill install wechat-article-monitor

# 添加公众号
/公众号监控 添加 36 氪

# 查看最新文章
/公众号监控 最新 36 氪

# 启动实时监控
/公众号监控 启动

# 导出文章
/公众号监控 导出 36 氪 markdown

# 列出所有监控的公众号
/公众号监控 列表

# 停止监控
/公众号监控 停止

# 删除公众号
/公众号监控 删除 36 氪
```

## 功能说明

### 1. 添加公众号

添加需要监控的公众号：

```
/公众号监控 添加 [公众号名称]
```

**示例**：
```
/公众号监控 添加 36 氪
/公众号监控 添加 极客时间
```

### 2. 查看最新文章

查看某个公众号的最新发布的文章：

```
/公众号监控 最新 [公众号名称]
```

**输出示例**：
```
📋 最新文章：

1. 《AI 时代，程序员如何保持竞争力？》
   📅 2026-03-16 11:00
   👁 12580 阅读
   👍 892 点赞
   🔗 https://mp.weixin.qq.com/s/xxx

2. 《2026 年科技行业趋势预测》
   📅 2026-03-16 09:30
   👁 10823 阅读
   👍 6421 点赞
   🔗 https://mp.weixin.qq.com/s/xxx
```

### 3. 启动实时监控

启动自动监控，定期检查新文章：

```
/公众号监控 启动
```

**说明**：
- 每小时检查一次（可配置）
- 发现新文章时推送通知
- 支持多公众号同时监控

### 4. 导出文章数据

将某个公众号的所有文章导出为文件：

```
/公众号监控 导出 [公众号名称] [格式]
```

**支持格式**：
- `markdown` - Markdown 格式，适合阅读
- `csv` - CSV 格式，适合数据分析

**示例**：
```
/公众号监控 导出 36 氪 markdown
/公众号监控 导出 极客时间 excel
```

### 5. 列出所有监控的公众号

查看当前监控的所有公众号：

```
/公众号监控 列表
```

**输出示例**：
```
📋 监控的公众号：

1. 36 氪
   ID: 36kr
   添加时间：2026-03-16 10:00
   文章数：5

2. 极客时间
   ID: geektime
   添加时间：2026-03-16 10:15
   文章数：5
```

### 6. 停止监控

停止所有正在进行的监控任务：

```
/公众号监控 停止
```

## 高级用法

### 定制检查间隔

修改检查间隔（默认 1 小时）：

```
# 编辑配置文件
openclaw config edit wechat-article-monitor

# 设置检查间隔为 30 分钟
checkInterval: 1800
```

### 通知设置

启用或禁用通知推送：

```
# 启用通知
notifyEnabled: true

# 禁用通知
notifyEnabled: false
```

### 文章数量限制

设置每次获取的最大文章数：

```
# 设置为 20 篇
maxArticles: 20
```

## 注意事项

### 使用限制
- 免费版最多监控 3 个公众号
- 免费版每次最多获取 5 篇文章
- 实时监控功能需要启用 `notifyEnabled`

### 数据存储
- 所有数据保存在本地 `./data` 目录
- 建议定期清理旧数据释放空间

### 提升建议
1. 对于重要的公众号，设置更短的检查间隔（30 分钟）
2. 定期备份 `./data` 目录
3. 关注检查日志，及时发现异常

## 常见问题

### Q: 提示"公众号不在模拟列表中"怎么办？
A: 这是正常的提示，技能会自动返回默认信息，不影响使用。

### Q: 如何查看已保存的文章？
A: 导出功能会将文章保存为 Markdown 或 CSV 文件，可以直接打开查看。

### Q: 监控会不会错过文章？
A: 如果设置了合理的检查间隔（如 1 小时），一般不会错过。如需实时监控，可缩短检查间隔。
