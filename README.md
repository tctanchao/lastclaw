## 🚀 快速开始

### 安装技能

```bash
# 克隆仓库
git clone https://github.com/tctanchao/lastclaw.git
cd lastclaw

# 安装技能（以小红书采集为例）
openclaw skill install ./skills/xiaohongshu-collector
```

### 开发新技能

```bash
# 创建技能模板
openclaw skill create my-new-skill

# 测试技能
openclaw skill test ./skills/my-new-skill
```
