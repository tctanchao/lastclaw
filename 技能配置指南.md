# 技能配置指南

## 🔧 需要安装的工具

### 1. GitHub CLI (gh)
**用途**: 查看PR、issue、CI状态，无需打开浏览器

**安装方法**:
```powershell
# 方法1: 使用winget
winget install --id GitHub.cli --source winget

# 方法2: 手动下载
# https://github.com/cli/cli/releases/latest
# 下载 gh_x.x.x_windows_amd64.msi 安装
```

**配置**:
```powershell
# 登录GitHub账户
gh auth login

# 验证
gh auth status
```

---

### 2. Spotify CLI (spogo)
**用途**: 控制Spotify播放、搜索歌曲

**安装方法**:
```powershell
# Windows暂无官方包，需要从源码编译或使用其他方案
# 替代方案: 使用 spotify_player
# 下载: https://github.com/aome510/spotify-player/releases

# 或者使用 scoop
scoop install spotify-player
```

**配置**:
```bash
# 导入浏览器cookie（需要Chrome登录过Spotify）
spogo auth import --browser chrome

# 或手动配置
# 编辑 ~/.config/spotify-player/app.toml
```

**注意**: 需要 Spotify Premium 账户

---

### 3. nano-pdf
**用途**: 用自然语言编辑PDF

**安装方法**:
```powershell
# 需要Python和pip
pip install nano-pdf

# 或使用 uv
uv install nano-pdf
```

**使用示例**:
```bash
nano-pdf edit document.pdf 1 "将标题改为Q3报告"
```

---

### 4. Notion API
**用途**: 创建和管理Notion页面、数据库

**无需安装CLI，只需配置API密钥**

**配置步骤**:
1. 访问 https://www.notion.so/my-integrations
2. 点击 "New integration"
3. 复制 API Key（以 `ntn_` 或 `secret_` 开头）
4. 在你的Notion页面中，点击 "..." → "Connect to" → 选择你的集成

**保存密钥**:
```powershell
# 创建配置目录
mkdir -Force ~/.config/notion
# 保存API密钥
echo "ntn_your_key_here" > ~/.config/notion/api_key
```

---

### 5. Gemini CLI
**用途**: 调用Google Gemini模型

**安装方法**:
```powershell
# Windows暂无官方包
# 可以使用 Gemini API 直接调用

```
