## 环境准备

### 必需工具

```bash

node --version

# OpenClaw CLI
openclaw --version

# Git
git --version
```

### 初始化项目

```bash
# 创建技能目录
mkdir skills/my-skill
cd skills/my-skill

# 初始化 npm 项目
npm init -y

# 安装依赖
npm install axios cheerio
```

## 开发流程

### Step 1: 需求分析

明确以下问题：
- 解决什么问题？
- 目标用户是谁？
- 核心功能有哪些？
- 如何变现？

### Step 2: 技术选型

| 功能 | 推荐方案 |
|------|----------|
| HTTP 请求 | axios |
| HTML 解析 | cheerio |
| Excel 导出 | exceljs |
| 定时任务 | node-cron |
| 数据库 | sqlite3 |

### Step 3: 编码实现

```javascript
// src/index.js
module.exports = {
  name: 'my-skill',
  version: '0.1.0',
  
  handlers: {
    async run({ param1, param2 }) {
      // 你的逻辑
      return { result: 'success' };
    }
  }
};
```

### Step 4: 本地测试

```bash
# 运行测试
npm test

# 开发模式
openclaw skill test ./skills/my-skill
```

## 最佳实践

### 代码规范

```javascript
// ✅ 好的做法
async function fetchData(url) {
  try {
    const response = await axios.get(url);
    return response.data;
  } catch (error) {
    console.error('获取失败:', error.message);
    throw error;
  }
}

// ❌ 避免
function fetchData(url) {
  return axios.get(url).then(r => r.data); // 无错误处理
}
```

### 错误处理

```javascript
// 友好的错误提示
if (!config.apiKey) {
  throw new Error('请先配置 API Key：openclaw config set apiKey=xxx');
}
```

### 性能优化

```javascript
// 批量处理，避免频繁请求
async function batchProcess(items) {
  const results = [];
  for (let i = 0; i < items.length; i += 10) {
    const batch = items.slice(i, i + 10);
    const batchResults = await Promise.all(batch.map(process));
    results.push(...batchResults);
    // 避免限流
    await sleep(1000);
  }
  return results;
}
```

## 常见问题

### Q: 技能无法加载？
A: 检查 SKILL.md 格式是否正确，package.json 是否有 main 字段

### Q: 如何调试？
A: 使用 `openclaw skill test --debug` 命令

### Q: 如何更新技能？
A: 修改版本号，重新发布
